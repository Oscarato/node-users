Bienvenido al API para las Pruebas NODE. 

La documentación está con Swagger en http://localhost:3000/api-docs


Para probar, realiza los siguientes pasos: 

  1. Desde un cliente como PostMan. Registra un usuario con el endpoint POST http://localhost:3000/register, suministrando el nombre, correo, contraseña y rol (El rol es un número entre 1 y 2 donde 1 es administrador y 2 es un usuario común). (http://localhost:3000/api-docs/#/General/post_register)
  2. Una vez est√© creado, se debe loguear para obtener el JWT en el endpoint http://localhost:3000/token. (http://localhost:3000/api-docs/#/General/post_token)
  3. Con el token ya podrá consultar la lista de usuarios http://localhost:3000/users (Si el rol es administrador) o un usuario en particular http://localhost:3000/users/64ff56924a48a8ded443be54 
  4. El administrador podrá eliminar o editar un usuario especificando el id.
